Задание 1.

Вычисление площади фигур.

1. Напишите метод с именем area с одним параметром типа double и с именем radius.
   Метод должен возвращать значение типа double, представляющее площадь круга.
   Если радиус параметра отрицателен, верните -1.0, чтобы представить недопустимое значение.
   Формула для расчета площади круга: радиус * радиус * PI.
   Для PI используйте константу из класса Math, например, Math.PI

2. Напишите другой перегруженный метод с 2 параметрами x и y (оба типа double),
   где x и y представляют стороны прямоугольника.
   Метод должен вернуть площадь прямоугольника. Если один или оба параметра являются отрицательными,
   метод должен вернуть -1.0, чтобы указать недопустимое значение.
   Формула для расчета площади прямоугольника имеет вид x * y.

   Примеры вызова / результута:
   area(5.0); должен вернуть 78.53975
   area(-1); должен вернуть -1, так как параметр является отрицательным
   area(5.0, 4.0); должен вернуть 20,0 (5 * 4 = 20)
   area(-1.0, 4.0); должен вернуть -1, так как первый параметр отрицателен

Все методы должны быть объявлены как public static.

Задание 2.

1. Вычисление равенства чисел.

   Напишите метод printEqual с 3 параметрами типа int. Метод не должен ничего возвращать (void).
   Если один из параметров меньше 0, выведите  в консоль текст «Неверное значение».
   Если все числа равны, выведите текст «Все числа равны».
   Если все числа разные, выведите текст «Все числа разные».
   В противном случае выведите «Не все числа разные».

   printEqual (1, 1, 1); должен печатать текст «Все числа равны»
   printEqual (1, 1, 2); должен печатать текст «Не все числа разные»
   printEqual (-1, -1, -1); должен печатать текст «Неверное значение»
   printEqual (1, 2, 3); должен печатать текст «Все числа разные»

2. Печать дней недели.

   а) Напишите метод с именем printDayOfTheWeek, у которого есть один параметр с именем day и типом int.
   Метод не должен возвращать никакого значения (подсказка: void)
   Используя оператор switch, выведите "Понедельник", "Вторник", и т.д., "Воскресенье",
   если параметр int "day" равен 0, 1, и т.д., 6 соответственно,
   в противном случае метод должен вывести "Invalid day"

   б) Напишите метод с именем printDayOfTheWeek, у которого есть один параметр с именем day и типом String.
   Метод должен выполнять тоже действие, используя if-then-else вместо использования switch.
   Для преобразования строки в число используйте метод Integer.parseInt()
