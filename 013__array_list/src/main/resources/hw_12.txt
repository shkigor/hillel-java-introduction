Задание 1.
Напишите метод с именем reverse() с массивом int в качестве параметра.
Метод не должен возвращать никакого значения. Другими словами, метод может изменять параметр переданного массива.
В main() протестируйте метод reverse() и выведите массив как в обратном, так и в не обращенном виде.

Чтобы перевернуть массив, вы должны поменять местами элементы, чтобы первый элемент поменялся местами с последним и так далее.
Например, если массив равен {1, 2, 3, 4, 5}, то обратный массив равен {5, 4, 3, 2, 1}.
