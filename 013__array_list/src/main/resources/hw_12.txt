Задание 1.
Напишите метод с именем reverse() с массивом int в качестве параметра. Метод должен перевернуть массив в обратном порядке.
Метод не должен возвращать никакого значения. Другими словами, метод изменяет массив (переданный как копия ссылки в параметре метода).
В main() протестируйте метод reverse() и выведите массив в начальном виде и в обратном.

Чтобы перевернуть массив, вы должны поменять местами элементы, чтобы первый элемент поменялся местами с последним и так далее.
Например, если массив равен {1, 2, 3, 4, 5}, то обратный массив равен {5, 4, 3, 2, 1}.
