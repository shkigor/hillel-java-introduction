Задание 1.
Создайте программу, которая реализует простой мобильный телефон со следующими возможностями:
хранить, изменять, удалять и запрашивать имена контактов.

Создайте отдельный класс для контактов (имя и номер телефона).
Создайте класс (MobilePhone), содержащий ArrayList<Contacts> и имя владельца телефона.

С помощью Scanner добавьте меню, содержащее следующие действия:
 - выйти из программы,
 - распечатать список контактов,
 - добавить новый контакт,
 - обновить существующий контакт,
 - удалить контакт,
 - найти контакт по имени.

При добавлении или обновлении обязательно проверьте, существует ли уже контакт.
