Пакетом по умолчанию является неименованный пакет. Неименованный пакет содержит классы Java, исходные файлы которых не содержат объявление пакета.
Неименованные пакеты (без имени) предоставляются платформой Java главным образом для удобства 
при разработке небольших или временных приложений или только в начале разработки.


В данном случае оба файла HelloFirstApp.java и HelloSecondApp.java находятся в одном каталоге.
Скомпилированные файлы классов .class создаются в текущей рабочей директории (в этом же каталоге, где и находятся файлы исходников .java)
