1. Скомпилируем исходные файлы командой 
   ```
   javac HelloHillel.java
   ```

2. Пробуем запустить используя команду 
   ```
   java HelloHillel
   ```

3. Теперь переместим Utils.class в другой каталог проекта, например BIN и попробуем запустить 
   ```
   java HelloHillel
   ```

4. Получаем ошибку. 
   Это произошло потому, что JRE не знает где искать класс Utils.
   JRE по умолчанию ищет необходимые классы в том же каталоге, что и запускаемый на выполнение класс.

5. Вот мы и подошли к важному понятию classpath.
   Так вот если мы укажем JRE где искать классы необходимые для работы классу HelloHillel.class, 
   то все сработает нормально.

6. Для этого используем ключ –classpath.
   ```
   java -classpath bin HelloHillel

   java -classpath .;bin HelloHillel
   ```
