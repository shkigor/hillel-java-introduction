1. Перейти в каталог где находятся папки hello и ua использую команду:
   ```
   cd папка_проекта
   ```
2. Откомпилировать класс HelloSecondApp:
   ```
   javac ua/ithillel/javabasic/lesson1/HelloSecondApp.java
   ``` 
3. Запустить:
   ```
   java ua.ithillel.javabasic.lesson1.HelloSecondApp
   ```
4. Откомпилировать класс HelloFirstApp: 
   ```
   javac hello/HelloFirstApp.java
   ```
5. Проверить/запустить: 
   ```
   java hello.HelloFirstApp
   ```
